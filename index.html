Type data pada database

1. Number / integer

  • INT : angka di bawah 2M
  • BIGINT : di atas 2M
  • DECIMAL :
    • DECIMAL(5, 2) → panjang number 5, dibelakang koma 2 → 999,99

2. String ( CHAR dan VARCHAR )

  • Maksimum char atay varchar adalah 65535 karakter
  • berbedaan :
    • misal char(4) maka tetap akan di masukkan 4 walaupun kosong → ‘ ‘
    • varchar fleksibel terhadap data yang di masukkan
  • TEXT
    • TINYTEXT max = 255
    • TEXT max = 65535
    • MEDIUMTEXT max = 16777215
    • LONGTEXT max = 4294967295
  • ENUM
    • digunakan untuk membuat pilihan
    • misal : data kelamain → hanya bisa ( ‘pria’ , ‘wanita’ )

3. Data date and time

  • DATE : YYYYY-MM-DD
  • DATATIME : tanggal dan waktu → YYYYY-MM-DD HH:MM:SS
  • TIME : waktu
  • YEAR : tahun

4. BOOLEAN

type data pengecekan apakah true jika benar dan false jika salah

  • ONE TO ONE RELATION

    tiap adata hanya bileh punya 1 data relasi

    CREATE TABLE wallet (
	id int NOT NULL PRIMARY KEY AUTO_INCREMENT,
	id_costomers int NOT NULL,
	balance int NOT NULL DEFAULT 0,
	UNIQUE KEY id_customer_unique (id_costomers),
	FOREIGN KEY fk_wallet_customers (id_costomers) REFERENCES customers(id)
);

SELECT customers.email, wallet.balance
FROM wallet JOIN customers ON (wallet.id_costomers = customers.id);
